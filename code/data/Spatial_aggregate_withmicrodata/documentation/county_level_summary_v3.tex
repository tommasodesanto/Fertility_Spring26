\begin{abstract}
This report presents a comprehensive analysis of fertility patterns at the county level, using restored and updated methodologies. It examines the relationship between housing costs (both Price-to-Income and Rent-to-Income ratios) and various fertility outcomes, controlling for demographic and economic factors. The analysis is divided into three main tracks: an aggregate-level panel data analysis, an individual-level analysis using microdata with contextual county variables, and a limited migration analysis. This version restores the full time-varying effects analysis and introduces a "horse race" comparison between contextual and individual-level housing cost burdens.
\end{abstract}
