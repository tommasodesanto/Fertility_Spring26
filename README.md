# Fertility And Housing Project

Current layout:

- `code/model/`: active Python model code.
- `code/cluster/`: active Torch calibration launcher and latest collected
  Python calibration result.
- `code/data/`: data construction and empirical target inputs.
- `code/empirical/`: ancillary empirical scripts.
- `latex/`: active writeups and slides.
- `docs/`: model notes, prompts, superpowers material, and archives.
- `output/`: generated outputs, with active model figures/results under
  `output/model/` and legacy spatial-wage outputs under
  `output/spatial_wage_analysis/`.
- `calibration_archive/`: archived logs, old MATLAB code, historical result
  folders, and deprecated status material.

Live calibration state is in `CALIBRATION_STATUS.md`. The former dated
workhorse folder has been flattened into the stable locations above.
