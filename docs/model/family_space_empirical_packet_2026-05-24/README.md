# Family-Space Empirical Packet

This folder is the stable, portable figure packet for the current family-space
empirical angle. It gathers the comparison figures and generated summary notes
that were previously scattered across ignored `output_*` folders.

## Main File

- `sample_comparison.html`: side-by-side visual comparison of the original MMS
  large-metro sample and the expanded top-100-c25 sample.

## Sample Definitions

- Original MMS: large regions with 2000 population at least 1,000,000 and a
  50 percent center-PUMA cutoff. Final fertility-supply sample: 42 metros.
- Expanded top-100-c25: top 100 CBD regions by 2000 population and a 25 percent
  center-PUMA cutoff. Final fertility-supply sample: 80 metros.

## Interpretation

The robust visual object is spatial incidence, not metro-wide fertility decline.
In both samples, central family-space scarcity predicts larger parent and
fertility gaps between periphery and center. Neither sample shows a robust
negative relationship between central family-space scarcity and metro-total
recent-birth rates.

## Contents

- `figures/`: copied PNG assets used by `sample_comparison.html`.
- `generated_summaries/`: copied markdown outputs from the original and
  expanded empirical packets.

The original reproducible outputs remain under
`code/data/mms_center_periphery/output_*`; those folders are generated and
ignored by Git.
