# Prompts

One-off review, audit, and testing prompts live here instead of the repository
root. They are useful context, but they are not active model or calibration
state.

- `claude_capability_test_2026-06-09/`: microfolder for testing a new Claude
  model on the current theory artifact, active Python model, calibration system,
  and project organization.
- `intergen_paper_expansion_handoff_20260708.md`: handoff prompt for expanding
  the current paper draft with the quantitative model, solution, calibration,
  diagnostics, and policy sections while fixing the main correctness issues.
